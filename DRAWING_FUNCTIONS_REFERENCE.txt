## DRAWING OUTPUT MODES (from DrawNote & others)

toScreen       — QuickDraw to screen (uses d2p for pixels)
toBitmapPrint  — QuickDraw to bitmap printer
toPICT         — QuickDraw to PICT resource
toPostScript   — PostScript output (uses DDIST directly)

Each function checks (switch outputTo) to decide coordinate system and API calls.


## VISIBILITY OPTIMIZATION

Critical variable in DrawScoreRange/DrawFormatRange/DrawMasterRange:
    Boolean drawAll
    
Purpose: Reduces redrawing of "spanning" objects that cross measure boundaries.

Set by checking:
    SectRect(&pMeasure->measureBBox, &paperUpdate, &result)
    
If no intersection (and outputTo == toScreen): drawAll = False
Otherwise: drawAll = True

Objects that check drawAll before drawing:
    PSYMEAS, CLEF, KEYSIG, TIMESIG, SYNC, GRSYNC, BEAMSET, TUPLET, OTTAVA, DYNAM, RPTEND, ENDING, GRAPHIC, TEMPO, SPACER, SLUR


## CONTEXT UPDATES

At PAGE:    GetAllContexts(doc, context, measL); ContextPage(doc, pL, context);
At SYSTEM:  Already set; may adjust via ContextSystem()
At STAFF:   Updates context[staffn] with staff-specific values (lines 614-623)
At MEASURE: Updates context[staff] with measure-specific values (lines 2810-2822)

Each level accumulates absolute page-relative DDIST positions.
